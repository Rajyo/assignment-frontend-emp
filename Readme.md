## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/Rajyo/assignment-frontend-emp.git
```
### 2. Change directory
```bash
cd assignment-frontend-emp
```

### 3. Install Dependencies
```bash
npm install
npm run dev
```

### 4. Run App
```bash
npm run dev
```

### 4. Access App
```bash
The application will be available at `http://localhost:5173`.
```
